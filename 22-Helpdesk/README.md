# WF-22 — Helpdesk Automation

Multi-channel helpdesk system built with n8n. Accepts support tickets from three sources (website form, Telegram, Gmail), saves them to Google Sheets, notifies the agent via Telegram, and sends auto-replies to users.

---

## Workflow files

| File | Trigger | Description |
|------|---------|-------------|
| `WF-22a-Helpdesk-Webhook.json` | POST Webhook | Receives tickets from a website contact form |
| `WF-22b-Helpdesk-Digest.json` | Schedule (09:00) | Sends a daily open-ticket summary to Telegram |
| `WF-22c-Help-Telegram.json` | Telegram message | Receives tickets sent directly to the bot |
| `WF-22d-Helpdesk-Gmail.json` | Gmail (poll) | Receives tickets from incoming emails |

---

## How it works

```
Website Form ──┐
Telegram Bot ──┼──► Normalize Data ──► Google Sheets ──► Notify Agent (Telegram)
Gmail Inbox ───┘                                        └──► Auto-reply to user

Schedule ──► Read Sheets ──► Build Digest ──► Telegram
```

Each sub-workflow follows the same pattern:
1. **Trigger** — catches the incoming message
2. **Normalize Ticket Data** — maps fields to a unified structure (`source`, `name`, `email`, `message`, `ticket_id`)
3. **Save Ticket to Sheets** — appends a row with `Status: new`
4. **Notify Agent** — sends a Telegram message to the support agent
5. **Auto-reply** — sends a confirmation back to the user (Telegram / Gmail)

---

## Google Sheets structure

Sheet name: **Tickets**

| Column | Description |
|--------|-------------|
| ID | Telegram chat_id (Telegram source only) |
| Timestamp | ISO 8601 creation time |
| Source | `website` / `telegram` / `email` |
| Name | Sender name |
| Email | Sender email |
| Message | Ticket text |
| Status | `new` → `in_progress` → `resolved` |

## Setup

### 1. Credentials to configure

After importing, replace all placeholder values with your real credentials:

| Placeholder | What to replace with |
|-------------|----------------------|
| `YOUR_CREDENTIAL_ID` | Your n8n credential ID (auto-assigned on import) |
| `YOUR_SPREADSHEET_ID` | Google Sheets document ID from the URL |
| `YOUR_TELEGRAM_CHAT_ID` | Your personal Telegram chat ID (get it from [@userinfobot](https://t.me/userinfobot)) |
| `YOUR_WEBHOOK_ID` | Auto-generated by n8n on import |
| `YOUR_INSTANCE_ID` | Your n8n instance ID |

### 2. Required n8n credentials

- **Google Sheets OAuth2** — connect your Google account
- **Telegram Bot API** — create a bot via [@BotFather](https://t.me/BotFather), paste the token
- **Gmail OAuth2** — connect the Gmail account that receives support emails

### 3. Import order

Import all four files into n8n. Activate them in this order:

```
22a → 22c → 22d → 22b
```

### 4. Website form (22a)

Point your contact form's POST request to the webhook URL:

```
POST https://your-n8n-instance.com/webhook/helpdesk-form

Body (JSON):
{
  "name": "User Name",
  "email": "user@example.com",
  "message": "Support request text"
}
```

## Daily Digest (22b)

Runs every day at **09:00** (server time). Sends a summary to the agent Telegram chat:

```
📊 Daily Helpdesk Report — 04.06.2026

🎫 Open tickets: 3 of 12

📡 By channel:
  • website: 1
  • telegram: 2
```
