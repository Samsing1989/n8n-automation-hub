# WF-26 — Freelancer Hiring

Automated freelancer hiring pipeline built with n8n. Accepts applications via webhook, scores resumes with AI (Groq / LLaMA), saves candidates to Google Sheets and HubSpot, notifies the HR manager, and schedules interviews via a Telegram bot command.

---

## Workflow files

| File | Trigger | Description |
|------|---------|-------------|
| `WF-26a-Intake-Pipeline.json` | POST Webhook | Receives applications, scores with AI, saves to Sheets + HubSpot, notifies HR |
| `WF-26b-Interview-Scheduler.json` | Telegram Bot command | Schedules interviews, sends email invite to candidate, sends reminder |

---

## How it works

### 26a — Intake Pipeline

```
POST /hr-intake
    │
    ▼
Prepare Candidate Data  ──►  AI Resume Scoring (Groq LLaMA)
                                    │
                                    ▼
                            Parse AI Response
                                    │
                          ┌─────────┴─────────┐
                          ▼                   ▼
                  Add to HR Pipeline    Confirm to Candidate (Gmail)
                  (Google Sheets)
                          │
                  ┌───────┴────────┐
                  ▼                ▼
        Create/Update Contact   Create Deal
           (HubSpot)             (HubSpot)
                  │
                  ▼
        Associate Contact to Deal
                  │
          ┌───────┴──────────────┐
          ▼          ▼           ▼
    Notify HR    Telegram HR   Respond to Webhook
    (Gmail)       (HR chat)    (JSON response)
          │
          ▼ (if AI score ≥ 6)
    Send Test Task (Gmail → Candidate)
```

### 26b — Interview Scheduler

```
Telegram Bot message
    │
    ▼
Check Command (/interview CAND-ID YYYY-MM-DD HH:MM)
    │                         │
    ▼ (valid)                 ▼ (invalid)
Parse Interview Command    Error message to user
    │
    ▼
Find Candidate (Google Sheets)
    │
    ▼
Update Status → interview_scheduled
    │
    ▼
Gmail → Candidate (interview invite)
    │
    ▼
Telegram → HR (confirmation)
    │
    ▼
Wait (1 hour)
    │
    ▼
Telegram → HR (reminder)
```

## Google Sheets structure

Sheet name: **Candidates**

| Column | Description |
|--------|-------------|
| candidate_id | Auto-generated `CAND-{timestamp}` |
| timestamp | ISO 8601 submission time |
| name | Candidate full name |
| email | Candidate email |
| phone | Candidate phone |
| position | Applied position |
| experience | Work experience description |
| skills | Skills list |
| ai_score | AI score 1–10 (Groq) |
| ai_summary | AI evaluation summary |
| ai_strengths | Candidate strengths |
| ai_concerns | Candidate concerns |
| ai_recommendation | AI hiring recommendation |
| status | `Новий` → `interview_scheduled` → ... |
| interview_date | ISO datetime of scheduled interview |

## Setup

### 1. Replace all placeholders

| Placeholder | Replace with |
|-------------|-------------|
| `YOUR_CREDENTIAL_ID` | Auto-assigned by n8n after connecting credentials |
| `YOUR_SPREADSHEET_ID` | Google Sheets document ID from the URL |
| `YOUR_TELEGRAM_CHAT_ID` | Your HR Telegram chat ID (get from [@userinfobot](https://t.me/userinfobot)) |
| `YOUR_HR_EMAIL@example.com` | HR manager email address |
| `YOUR_WEBHOOK_ID` | Auto-generated by n8n on import |
| `YOUR_INSTANCE_ID` | Your n8n instance ID |
| `YOUR_VERSION_ID` | Auto-assigned by n8n |

### 2. Required credentials

| Credential | Used in |
|------------|---------|
| **Google Sheets OAuth2** | Both workflows — read/write candidates |
| **Gmail OAuth2** | Both workflows — send emails |
| **Telegram Bot API** | Both workflows — bot trigger + notifications |
| **Groq API** (HTTP Header Auth) | 26a — AI resume scoring |
| **HubSpot App Token** | 26a — create contacts and deals |
| **HubSpot Private App Token** (HTTP Header Auth) | 26a — associate contact to deal |

### 3. Import order

```
26a → 26b
```

Activate 26a first — it creates candidates in Sheets that 26b searches.

## Webhook — Intake Pipeline (26a)

**Endpoint:**
```
POST https://your-n8n-instance.com/webhook/hr-intake
```

**Body (JSON):**
```json
{
  "name": "John Doe",
  "email": "john@example.com",
  "phone": "+380991234567",
  "position": "Frontend Developer",
  "experience": "3 years of React, Vue.js",
  "skills": "JavaScript, TypeScript, CSS, Git",
  "resume_text": "Full resume text goes here..."
}
```

**Response:**
```json
{
  "success": true,
  "candidate_id": "CAND-1718123456789",
  "message": "Заявку прийнято"
}
```

## Telegram Bot — Interview Scheduler (26b)

**Command format:**
```
/interview CAND-ID YYYY-MM-DD HH:MM
```

**Example:**
```
/interview CAND-1718123456789 2026-07-10 14:00
```

**What happens:**
1. Bot finds the candidate in Google Sheets by ID
2. Updates status to `interview_scheduled`
3. Sends email invitation to candidate
4. Sends confirmation to HR in Telegram
5. Waits 1 hour → sends reminder to HR

---

## AI Scoring Logic

The Groq LLaMA-3.3-70b model scores each resume from **1 to 10** and returns:

| Field | Description |
|-------|-------------|
| `score` | Numeric score 1–10 |
| `summary` | Overall evaluation |
| `strengths` | Key candidate strengths |
| `concerns` | Potential concerns |
| `recommendation` | Hire / Consider / Reject |

Candidates with **score ≥ 6** automatically receive a test task via email.
